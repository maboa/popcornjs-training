Popcorn Plugins

Popcorn.js has a great pluginable architecture. This means that the core can stay nice and light and you only import the plugins you need. It is modular to its core which is a good thing.

So lets take a brief look at the various plugins that Popcorn officially supports.

  • Attribution
  • Code
  • Facebook
  • Flickr
  • Footnote
  • GML
  • Googlefeed
  • Googlemap
  • Image
  • Lastfm
  • Linkedin
  • Lowerthird
  • Manifest
  • Mustache
  • OpenMap
  • Pause
  • Processing
  • Subtitle
  • Tagthisperson
  • Timeline
  • Tumblr
  • Twitter
  • Webpage
  • Wikipedia
  • Wordriver

We'll look at some of these in more detail later and others that aren't on the list.
