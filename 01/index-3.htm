Flavours of Popcorn.js (1.3)

  • Popcorn Core (popcorn.min.js) is the core library - (7.2 kB zipped)
  • Popcorn Complete (popcorn.complete.min.js) is popcorn.js plus most supported plugins - (86.3 kB zipped)
  • Popcorn IE8 (popcorn-ie8.js) is a fallback which you need in addition to popcorn core to ensure IE8 compatibility - (10.1 kB zipped)
Note non-minified versions are available for debug purposes.

Popcorn folklore and legend : There used to be a web-based Popcorn builder, where you could specify the plugins you wanted to include and it would build a special version just for you.
