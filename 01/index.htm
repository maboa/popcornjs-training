How would you like your Popcorn?

Popcorn is a JavaScript Library (Popcorn.js) and a video editing tool (Popcorn Maker) which makes up part the Mozilla Foundation's Webmaker initiative

We're going to concentrate on Popcorn.js in this course, but first a brief lowdown on the wider Popcorn thing.

Popcorn Maker is the second major iteration of creating a web-based video editing tool for the masses. The first was called Butter Player. However Butter is actually an SDK for authoring Popcorn projects.

I think Popcorn Maker is a great tool, it's simple and powerful and newer versions (still in development I believe) are focussing on sequencing or remixing from various sources. Popcorn Maker is built with Popcorn.js and so allows the Popcorn Team to dogfood the core library.

So to sum up ...

  • Popcorn.js is the JavaScript library used chiefly by developers.
  • Popcorn Maker is a visual editor that uses Popcorn.js at its core and is used by anyone.
  • Butter Player is the prequel to Popcorn Maker - now largely defunct.
  • Butter is an SDK for authoring Popcorn projects.
